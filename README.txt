# Employee Class Example

This is a simple example of an employee management system implemented in C++ using class inheritance.

## Instructions

1. Clone or download the repository to your local machine.

2. Open a C++ compiler or IDE (e.g., Visual Studio, Code::Blocks, or a C++ compiler of your choice).

3. Add the source files (`employee.h`, `employee.cpp`, and `main.cpp`) to your project.

4. Build and run the project.

5. The program will create instances of professional and nonprofessional employees, calculate their weekly salary, healthcare contributions, and vacation days, and display the results.

6. Enjoy exploring the employee management system example!
